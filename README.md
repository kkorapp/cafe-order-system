# Cafe Order System

## Description
This project is a simple Java console application for a small cafe. It allows creating menu items, customers, and orders.

## Classes
- MenuItem
- CafeCustomer
- Order
- Main

## Functionality
- create menu items
- create a customer
- add products to an order using ArrayList
- calculate total order value
- print object summaries using toString()
- compare objects using equals()
- mark an order as paid
